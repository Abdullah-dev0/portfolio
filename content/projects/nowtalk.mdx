# NowTalk - Real-time Chat Platform

A modern real-time chat application that enables seamless communication and helps build dynamic online communities.

## Overview

NowTalk leverages Supabase's real-time capabilities to provide instant messaging, presence detection, and community features.

## Key Features

- **Real-time Messaging**: Instant message delivery with Supabase Realtime
- **User Presence**: See who's online in real-time
- **Channel Management**: Create and manage multiple chat channels
- **Rich Media Support**: Share images, files, and links
- **User Authentication**: Secure login with Supabase Auth

## Technology Stack

The application is built with modern web technologies:

- Next.js for the frontend framework
- Supabase for backend and real-time functionality
- TypeScript for type safety
- Tailwind CSS for styling
- shadcn/ui for UI components
